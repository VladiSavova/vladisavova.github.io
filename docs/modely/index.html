TESLA MODEL Y

The Tesla Model Y is an upcoming electric compact crossover utility vehicle (CUV) in development by Tesla, Inc. It was unveiled in March 2019, with deliveries starting in late 2020. It is Tesla's second vehicle built upon the tier-3 mass-market-vehicle platform, alongside the Model 3.
